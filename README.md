# CleanFrames

Minimalist World of Warcraft addon that hides auras and visual elements on arena frames, nameplates, and raid frames.

## Features

- **Arena Frames**: Hides names, debuffs, and cast bars on arena enemy frames
- **Nameplates**: Removes buff displays from enemy nameplates
- **Raid Frames**: Hides all buffs and debuffs on raid/party frames

## Installation

1. Download the addon
2. Extract to `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or reload UI with `/reload`

## Purpose

Reduces visual clutter in competitive PvP environments by removing distracting aura information from frames, allowing focus on core gameplay elements.

## Compatibility

- **Interface Version**: 11.2.0 (The War Within)
- **Conflicts**: May conflict with other raid frame or nameplate addons
